# PowerShell-EntraID-Groups
PowerShell scripts for Microsoft 365 and Entra ID group management — синтаксис и обяснения от Цено Стаменов.


### **1. Get-All-Groups.ps1**  
Извлича всички групи в Microsoft 365 / Entra ID — Security, Mail-enabled, Microsoft 365 и Distribution групи.

---

### **2. Get-All-Groups-Detailed.ps1**  
Показва всички групи с допълнителни атрибути като MailEnabled, SecurityEnabled и GroupTypes, за да се разпознае типът на всяка група.

---

### **3. Export-GroupMembers.ps1**  
Експортира всички членове на избрана група в CSV файл.

---

### **4. Get-DynamicGroups.ps1**  
Извежда само динамичните групи и показва техните правила за членство (MembershipRule).

---

### **5. Group-Types-CheatSheet.md**  
Кратка таблица с обяснения как да се разпознават различните типове групи в Microsoft 365 / Entra ID.
